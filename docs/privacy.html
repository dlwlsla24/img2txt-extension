개인정보처리방침

최종 업데이트: 2026년 4월 7일

1. 개요

쇼핑몰 이미지 텍스트 추출기는 쇼핑몰 상품 상세 페이지의 이미지에서 텍스트를 OCR(광학 문자 인식) 기술로 추출하는 Chrome 확장프로그램입니다.

본 방침은 이 확장프로그램이 수집하는 정보와 사용 방법을 설명합니다.

2. 수집하는 정보

✅ 이 확장프로그램은 개인 식별 정보를 수집하지 않습니다.

확장프로그램이 로컬에 저장하는 정보:

  • OCR API 키 — 사용자가 직접 입력한 API 키를 Chrome의 로컬 스토리지(chrome.storage.local)에 저장합니다. 이 데이터는 사용자 기기 외부로 전송되지 않습니다.
  • 선택한 OCR 엔진 — 사용자의 엔진 선택 설정(OCR.space, Google Cloud Vision, Tesseract.js)을 로컬에 저장합니다.

3. 외부 서비스 연동

사용자가 선택한 OCR 엔진에 따라 아래 외부 서비스에 이미지 데이터가 전송될 수 있습니다:

  • OCR.space API — OCR.space 엔진 선택 시, 이미지가 OCR.space 서버로 전송됩니다. OCR.space 개인정보처리방침
  • Google Cloud Vision API — Google Cloud Vision 엔진 선택 시, 이미지가 Google 서버로 전송됩니다. Google 개인정보처리방침
  • Tesseract.js — Tesseract.js 엔진 선택 시, 모든 처리가 사용자 기기에서 이루어지며 외부 서버로 데이터가 전송되지 않습니다.

⚠️ 이미지 외의 개인정보(계정 정보, 결제 정보 등)는 어떠한 외부 서버로도 전송되지 않습니다.

4. 권한 사용 목적

  • activeTab — 현재 탭의 쇼핑몰 페이지에서 이미지 URL을 추출하기 위해 사용됩니다.
  • scripting — 쇼핑몰 페이지에 이미지 탐지 스크립트를 실행하기 위해 사용됩니다.
  • storage — API 키 및 엔진 설정을 로컬에 저장하기 위해 사용됩니다.

5. 데이터 보안

API 키는 chrome.storage.local에만 저장되며:

  • 개발자 또는 제3자에게 전송되지 않습니다.
  • 사용자 기기에만 존재합니다.
  • 확장프로그램을 삭제하면 모든 저장 데이터가 삭제됩니다.
