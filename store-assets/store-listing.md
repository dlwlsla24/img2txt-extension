## 상세 설명 (한국어)

**쇼핑몰 상품 상세 이미지의 텍스트를 한 번에 추출하세요!**

쇼핑몰 상품 상세 페이지의 긴 이미지들에 들어있는 텍스트를 일일이 타이핑할 필요가 없습니다.
확장프로그램 아이콘 클릭 한 번으로 모든 이미지에서 텍스트를 자동으로 추출합니다.

**✨ 주요 기능**
• 상품 상세 이미지 자동 탐지 - 페이지의 모든 상세 이미지를 자동으로 찾아냅니다
• 한국어 OCR - 높은 정확도의 한국어+영어 텍스트 인식
• 3가지 OCR 엔진 선택 - OCR.space(무료 추천), Google Cloud Vision(최고 정확도), Tesseract.js(오프라인)
• 클립보드 복사 - 추출된 텍스트를 즉시 복사
• TXT 파일 다운로드 - 텍스트 파일로 저장
• 이미지별 개별 복사 - 원하는 이미지의 텍스트만 선택 복사
• 처리 중 탭 유지 - 추출 중 실수로 클릭해도 작업이 중단되지 않음

**🏪 지원 쇼핑몰**
올리브영, 쿠팡, 11번가, G마켓, 옥션, SSG, 마켓컬리, 무신사 + 기타 대부분의 쇼핑몰

월 25,000건 무료, 카드 등록 불필요!

## 상세 설명 (English)

**Extract text from shopping mall product images instantly!**

No more manual typing of text from long product detail images.
Click the extension icon once to automatically extract text from all product images.

**✨ Key Features**
• Auto image detection - Automatically finds all product detail images
• Korean OCR - High accuracy Korean + English text recognition
• 3 OCR engines - OCR.space (free, recommended), Google Cloud Vision (best accuracy), Tesseract.js (offline)
• Copy to clipboard - Instantly copy extracted text
• TXT file download - Save text as a file
• Per-image copy - Copy text from individual images
• Tab persistence - Extraction continues even if you accidentally click away

**🏪 Supported Sites**
Oliveyoung, Coupang, 11ST, Gmarket, Auction, SSG, Kurly, Musinsa + most Korean shopping malls

**🆓 Free OCR API**
Get a free OCR.space API key at ocr.space/ocrapi/freekey - just enter your email, no credit card required.
25,000 requests/month for free.

## 권한 사용 목적 설명 (심사용 - 개발자 대시보드 "권한 정당성" 란에 입력)

**activeTab**
사용자가 현재 보고 있는 쇼핑몰 페이지에서 상품 상세 이미지의 URL을 추출하기 위해 사용됩니다.
사용자가 확장프로그램 아이콘을 클릭한 시점에만 활성화됩니다.

**scripting**
쇼핑몰 페이지에 콘텐츠 스크립트를 주입하여 이미지 요소를 탐지하기 위해 사용됩니다.
이미지 URL 추출 외의 다른 목적으로는 사용되지 않습니다.

**storage**
사용자가 입력한 OCR API 키와 선택한 엔진 설정을 로컬에 저장하기 위해 사용됩니다.
저장된 데이터는 기기 외부로 전송되지 않습니다.
