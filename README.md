# 🔍 쇼핑몰 이미지 텍스트 추출기

쇼핑몰 상품 상세 페이지의 긴 이미지들에서 **텍스트를 자동으로 추출**하는 크롬 확장프로그램입니다.

## ✨ 주요 기능

- 🛒 **자동 이미지 탐지** - 상품 상세 페이지의 모든 설명 이미지를 자동으로 찾아냅니다
- 🇰🇷 **한국어 OCR** - Tesseract.js 기반 한국어 + 영어 텍스트 인식
- 📋 **텍스트 복사** - 추출된 텍스트를 클립보드에 바로 복사
- 💾 **파일 다운로드** - .txt 파일로 다운로드
- ☑️ **이미지 선택** - 원하는 이미지만 골라서 추출 가능

## 📦 설치 방법

### 1. 파일 준비
이 레포지토리를 클론하거나 다운로드합니다.

```bash
git clone https://github.com/your-username/igetnoroots.git
```

### 2. 크롬에 확장프로그램 설치

1. Chrome 브라우저에서 주소창에 `chrome://extensions` 입력
2. 우측 상단의 **"개발자 모드"** 토글을 켭니다
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. 다운로드한 `igetnoroots` 폴더를 선택

![설치 완료 후 확장프로그램 아이콘이 툴바에 나타납니다](docs/install.png)

## 🚀 사용 방법

1. **쇼핑몰 상품 페이지로 이동** - 올리브영 등 지원 쇼핑몰의 상품 상세 페이지를 엽니다
2. **확장프로그램 아이콘 클릭** - 툴바의 확장프로그램 아이콘을 클릭합니다
3. **이미지 자동 탐지** - 상세 이미지가 자동으로 탐지되어 목록에 표시됩니다
4. **텍스트 추출 시작** - "텍스트 추출 시작" 버튼을 클릭합니다
5. **결과 확인** - 추출된 텍스트를 복사하거나 .txt 파일로 다운로드합니다

## 📁 프로젝트 구조

```
igetnoroots/
├── manifest.json      # 크롬 확장프로그램 설정
├── popup.html         # 팝업 UI
├── popup.css          # 팝업 스타일
├── popup.js           # 팝업 로직 (OCR 처리)
├── content.js         # 콘텐츠 스크립트 (이미지 추출)
├── icons/             # 확장프로그램 아이콘
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── libs/              # Tesseract.js 라이브러리
    ├── tesseract.min.js
    ├── worker.min.js
    └── tesseract-core.wasm.js
```

## ⚙️ 기술 스택

- **Chrome Extension Manifest V3**
- **Tesseract.js v5** - 브라우저 기반 OCR 엔진
- **한국어 학습 데이터** - tessdata 4.0.0 (kor+eng)

## ⚠️ 참고사항

- OCR 정확도는 이미지 품질에 따라 달라질 수 있습니다
- 첫 실행 시 한국어 학습 데이터를 다운로드하므로 시간이 약간 걸릴 수 있습니다
- 이미지가 많을수록 처리 시간이 길어집니다
- 추출된 텍스트는 개인 용도로만 사용해 주세요
